# 天梯最后一级

在塔顶铺设最后一级临时天梯，用剩余材料让一队人安全通过。

## 在线试玩

https://dengxiaocheng.github.io/BabelMicrogame-TiantiZuihouYiji/

## 本地运行

```bash
npm start    # 启动本地服务器 (http://localhost:8080)
npm test     # 运行测试
```

## 操作说明

1. **搭建阶段** — 拖放板 / 绳 / 楔到桥面缺口
2. **排序阶段** — 拖动人员安排通过顺序
3. **通过阶段** — 观察结构受力与风力变化
4. 成功让所有人通过即为胜利

Babel microgame source repository.
