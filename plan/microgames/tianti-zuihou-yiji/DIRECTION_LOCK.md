# Direction Lock: 天梯最后一级

## One Sentence
玩家在塔顶铺设最后一级临时天梯，必须用剩余材料让一队人短暂通过。

## Core Loop
1. 执行核心循环：查看缺口 -> 放置板/绳/楔 -> 安排通过顺序 -> 结构受力 -> 成功通过或坠落
2. 按 20 分钟节奏推进：短缺口 -> 多材料 -> 人员重量差 -> 塔顶风暴下最后通行

## Must Keep
- 核心机制必须保持：临时桥搭建 + 承重顺序 + 人员通行
- 核心循环必须保持：查看缺口 -> 放置板/绳/楔 -> 安排通过顺序 -> 结构受力 -> 成功通过或坠落
- 20 分钟结构只作为节奏，不扩成大项目

## Must Not Add
- 不做终章剧情；核心是临时结构承重
- 不新增第二套主循环
- 不把小游戏扩成长期经营或开放世界

## Required State
- resource
- pressure
- risk
- relation
- round

## Success
在限定时长内完成主循环，并稳定进入至少一个可结算结局

## Failure
关键状态崩溃，或在本轮主循环中被系统淘汰
